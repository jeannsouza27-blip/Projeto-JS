
# 💱 Conversor de Moedas

Este é um projeto de **Conversor de Moedas** desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de praticar consumo de API, manipulação do DOM e criação de interfaces interativas.

A aplicação permite converter valores em **Real Brasileiro** para diferentes moedas, trazendo uma experiência visual simples, organizada e funcional.

## 🚀 Funcionalidades

- Conversão de moedas em tempo real
- Exibição do valor original e convertido
- Alteração dinâmica de nome, símbolo e bandeira da moeda
- Interface responsiva e intuitiva

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
- AwesomeAPI

## 📚 O que aprendi

Durante o desenvolvimento deste projeto, pratiquei:

- Estrutura semântica com HTML
- Estilização e organização visual com CSS
- Lógica de programação com JavaScript
- Manipulação de eventos
- Consumo de API com `fetch`
- Atualização dinâmica de elementos na tela

## 🎯 Finalidade

Este projeto faz parte da minha jornada de aprendizado em desenvolvimento front-end, buscando evoluir cada vez mais na criação de aplicações web modernas e funcionais.
